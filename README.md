# Smart IoT Energy Monitoring System

An IoT-based smart energy monitoring system developed using ESP32, ACS712 current sensor, PIR sensor, and the Blynk IoT platform.

## Features

- Real-time energy monitoring
- Occupancy detection using PIR sensor
- Remote monitoring using Blynk
- Basic automation for energy saving
- Serial monitor output for debugging and analysis

## Components Used

- ESP32
- ACS712 Current Sensor
- PIR Motion Sensor
- LED / Relay Module
- Blynk IoT Platform

## Technologies Used

- Arduino IDE
- ESP32
- Embedded C++
- Blynk IoT
- Proteus Simulation

## System Workflow

1. PIR sensor detects occupancy
2. ACS712 measures current consumption
3. ESP32 processes sensor data
4. Data is sent to the Blynk mobile application
5. System can automate load behaviour based on occupancy
